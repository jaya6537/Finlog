# FinLog Nexus (MERN Web)

This is the web conversion of your React Native FinLog app using MERN stack.

## Project Name
**FinLog Nexus**

## Stack
- Frontend: React (`.jsx`) + Vite + reusable components
- Backend: Node.js + Express + MongoDB
- Architecture: Backend follows MVC (`models`, `controllers`, `routes`)

## Folder Structure
- `frontend/` - React web app
- `backend/` - Express API in MVC style

## Run Instructions

### Backend
1. Open `backend/.env.example`, copy values into `backend/.env`
2. Install and run:
   - `cd backend`
   - `npm install`
   - `npm run dev`

### Frontend
1. Install and run:
   - `cd frontend`
   - `npm install`
   - `npm run dev`

Frontend runs on `http://localhost:5173` and talks to backend on `http://localhost:5000`.
